# FootIN App

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/rayanebahri23/footin-app)

Une application Flutter pour le recrutement dans le football amateur.

## Démarrage Rapide (Cloud)

1. Cliquez sur le bouton **"Open in GitHub Codespaces"** ci-dessus.
2. Attendez que l'environnement se charge.
3. Une fois prêt, l'application se lancera automatiquement sur le web.

Si elle ne se lance pas automatiquement, tapez :
```bash
flutter run -d web-server
```

## Fonctionnalités
- Marketplace de talents
- Profils vérifiés
- Recherche scout
